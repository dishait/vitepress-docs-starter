# vitepress-docs-template

基于 `vitepress` 的文档模板 👍

<br />

## 动机 🦖

为了快速书写漂亮的开源产品文档！！


<br />
<br />

## 特性 🦕

- [vueuse](https://vueuse.org/) 支持
- [inspect](https://github.com/antfu/vite-plugin-inspect) 调试支持
- [unocss](https://github.com/unocss/unocss) 原子库支持
